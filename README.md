# movi_search_website
Movie Search Website is a web application that allows users to search for movies, view details, and explore information like release date, ratings, genres, and synopsis. It uses a public movie API to fetch real-time data and provides a clean, user-friendly interface.

This project is perfect for learning API integration, front-end development, and responsive UI design.
